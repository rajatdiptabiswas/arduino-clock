# Arduino Clock
A seven segment display clock made using Arduino Uno

## Hardware used
* Seven segment displays (Common Anode LT542) x 4
* Bread board x 1
* Push buttons x 4
* 1k Î© resistors x 4
* Arduino Uno x 1
* Jumper wires
